## Installation

### Server Requirements

The Lavalite framework has a few system requirements. 

    - PHP >= 7.2
    - OpenSSL PHP Extension
    - PDO PHP Extension
    - Mbstring PHP Extension
    - Tokenizer PHP Extension
    - Fileinfo PHP Extension
    - GD Library
    - Imagick PHP Extension
